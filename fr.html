    • FreeBet

    Une application web PHP pour réaliser des paris sportifs facilement et gratuitement

    Projet maintenu par jbouzekri Hébergé sur GitHub Pages — Theme par mattgraham

    Qu'est ce que FreeBet ?

    FreeBet vous permet de tester vos capacités de parieur sportif.

    • C'est totalement gratuit.
    • Il n'y a pas d'argent en jeu donc vous ne pouvez rien perdre.
    • Vous pouvez vous entraîner avant de passer sur des plateformes de pari à mises réelles
    • C'est opensource. Vous pouvez le télécharger, l'installer (il est utilisé dans mon entreprise) et contribuer.

    Comment ça marche ?

    Vous pouvez faire vos paris sur différents évènements sportifs. Pour le moment, seul la coupe du monde 2014 est couverte mais je vais rajouter rapidement les ligues de football européennes. Quand un évènement est terminé, un administrateur saisit le résultat final. Une tache est exécutée pour déterminer si votre pari est gagnant ou non. Vous gagnez des points pour chaque pari réussi.

    • Un pari simple vous donne 1 point
    • Un pari plus complexe vous donne 15 point (par exemple le score exact d'un match de footbal)
    • Des paris multiples réussis multiplient le nombre total de points gagnés
    Ce n'est pas encore implémenté mais les joueurs apparaîtront prochainement dans un classement mondial.
