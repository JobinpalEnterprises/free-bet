    A PHP application to provide fun and free betting on sport's events

    Project maintained by Jonathan Bouzekri Hosted on GitHub Pages — Theme by mattgraham

    What is FreeBet ?

    FreeBet provides a way to test your capability as a forecaster on sport's events.

    • This is a full sports forecast platform totaly free.
    • There is no money involved so you cannot lose anything by playing.
    • This can be a good way to start your sports forecast before moving on paying platforms.
    • FreeBet is an open source software. You can download it, install it (it is running in my company) and contribute to it.

    How does it work ?

    You can make your gambles on the different sports events. For starter, there is only the 2014 world cup but I am going to add all the european leagues very soon. When an event ends, an admin sets the final score and a task processes your gambles to check if it is a winner or not. Payers earn points for each successful gambles

    • A simple forecast gives you 1 point
    • A difficult one gives you 15 points (for example score exact in soccer)
    • Multiple bets in the same gamble multiply the total by the number of bets
    This is not yet implemented but players will be ranked in the future.
